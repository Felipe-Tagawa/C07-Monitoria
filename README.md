# C07 - Banco de Dados (Monitoria)
Este repositório contém os materiais, scripts e exercícios desenvolvidos para o suporte à disciplina de Banco de Dados (C07) no Inatel. O objetivo é fornecer exemplos práticos de modelagem, manipulação de dados e consultas SQL.

Conteúdo do Repositório
O projeto está organizado para cobrir as principais etapas de gerenciamento de um banco de dados relacional:

DDL (Data Definition Language): Scripts para criação de esquemas, tabelas, definição de chaves primárias, estrangeiras e restrições (constraints).

DML (Data Manipulation Language): Comandos para inserção, atualização e exclusão de registros para popular o banco de testes.

DQL (Data Query Language): Consultas SQL, abrangendo desde seleções simples até junções complexas (JOINs), agrupamentos e subconsultas.

DCL (Data Control Language): Controle de Acesso de Usuários ao Banco de Dados criado e implementado.

Structures (Procedures, Functions, Views e Triggers): Estruturas de Programação do Banco de Dados.

Integração com Python e Java: Utilização de Maven e como integrar um banco de dados ao Backend estruturado nas linguagens de alto nível.

Como Utilizar
Clone o repositório:
git clone https://github.com/Felipe-Tagawa/C07-Monitoria

Execute o script de criação do banco de dados em seu ambiente local (MySQL Workbench).

Utilize os scripts de inserção para gerar dados de teste.

Pratique com os exercícios de consulta disponibilizados.
